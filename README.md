## 🚀 The Project
This repo covers two main areas:
1. **Software Agency Landing Page:** A modern, conversion-focused site.
2. **"Orbilex Market" E-commerce Platform:** A full-featured store and admin dashboard.

### 🛒 Customer Storefront ("Orbilex Market")
* **Dynamic Product Browsing:** Seamless filtering by category (Produce, Dairy, etc.).
* **Interactive Cart:** Real-time state management for items, quantities, and instant tax/subtotal calculation.
* **Responsive UI:** Fully optimized for mobile and desktop.

### ⚙️ Store Manager / Admin Dashboard
* **Analytics:** At-a-glance revenue trends, order tracking, and performance metrics.
* **Inventory Management:** Real-time low-stock alerts.
* **Full CRUD:** Complete control to add, edit, or remove products and categories.
